## 🎯 Part A 概述

### 報告定位

券商風格深度研究報告：
- 目標讀者：專業投資人、機構投資者
- 閱讀時間：10-15 分鐘
- 總字數：3,000-3,500 字（±10%）
- 風格：客觀、數據驅動、邏輯嚴謹

### 核心要求

| 維度 | 要求 |
|-----|------|
| **內容深度** | 詳細分析邏輯 + 數據支撐 + 量化風險 |
| **語言風格** | 自然語言，避免理論術語和過度學術化 |
| **格式控制** | 散文段落為主，嚴格字數限制 |

## 📝 區塊 1：重要訊息（80-120字）

### 結構公式

```
[股價現況20字] + [核心論點40字] + [主要風險30字] + [分析師態度25字]
```

### 必須包含

- 當前股價、目標價、上漲/下跌空間
- 核心投資論點（1-2句）
- 主要風險（1句）
- 分析師態度

### 格式要求

- 單一散文段落（非條列式）
- 嚴格控制在 80-120 字

### 優秀範例

```markdown
## 重要訊息

AMD 當前股價 US$198.11，相對公允價值估計 US$270.00 仍有 36%
上漲空間。公司在 AI 晶片市場取得重大突破，與 OpenAI 建立戰略
合作夥伴關係，預期 2026 下半年開始供應 MI450 GPU 產品。然而，
Nvidia 在 AI GPU 市場的主導地位與 CUDA 軟體生態系統優勢仍是
AMD 面臨的最大挑戰。我們看好 AMD 在資料中心市場的長期增長
潛力，但考量 Very High 不確定性評級，投資人需密切關注軟體
能力提升進展。
```

## 📊 區塊 2：評論及分析（三個子章節）

### 章節劃分

| 項目 | 要求 |
|-----|------|
| 章節數量 | 固定 3 個 |
| 每章字數 | 300-500 字 |
| 段落結構 | 每章 2-3 段 |
| 內容來源 | 階段0C選定的3-5個分析視角 |

### 段落結構模板

```
第1段（120-180字）：現象 + 數據
  → 陳述現象/趨勢 + 具體數據 + 量化影響

第2段（120-180字）：原因 + 機制
  → 解釋原因 + 影響機制 + 因果邏輯

第3段（60-140字）：展望 + 風險（可選）
  → 預測趨勢 + 潛在風險 + 時間框架
```

---

### 子章節 1：營運表現

#### 內容要求（整合同事模板）

**建議採用三段落結構**（非強制，可依數據彈性調整）：

1. **第一段：核心財務表現**
   - 關鍵財務數據（營收、利潤、EPS）
   - 同比/環比增長率
   - 毛利率、營業利益率變化

2. **第二段：重大戰略調整**
   - 資產剝離或收購（M&A）
   - 業務重組
   - 策略轉型

3. **第三段：管理層財測指引**
   - 下季度/年度展望
   - 財務目標
   - 業務驅動因素

#### 章節標題範例

✅ 推薦：
- "AI GPU 業務迎來爆發性增長"
- "資本運用效率持續提升"
- "Q3 業績超越市場預期"

❌ 禁止：
- "營運表現分析"（過於制式）
- "第一節：財務分析"（編號式）

---

### 子章節 2：市場展望

#### 內容要求（整合同事模板）

**必須包含**：
1. **營收增長主力分析**
   - 主要產品/服務增長動力
   - 市場需求趨勢

2. **競爭分析**
   - 主要競爭對手
   - 競爭優勢與市場地位

3. **季度變化比較（新增）**
   - 本季 vs 上季關鍵變化
   - 市場份額變動
   - 競爭態勢演變

#### 季度變化分析格式

```markdown
### [標題]

[第1段：當前市場狀況]

[第2段：競爭格局與公司地位]

相較上季度（QX），本季度（QY）呈現以下關鍵變化：[產品需求趨勢變化]
[競爭對手動態]，[公司競爭優勢的強化/削弱跡象]。
```

#### 資料來源

- Morningstar 季度研究報告
- 近期產業文章（限 ANALYSIS_START_DATE ~ ANALYSIS_END_DATE）
- 公司季報與法說會資料

---

### 子章節 3：估值與目標價

#### 內容要求（整合同事模板）

**必須包含**：
1. **EPS 推估依據**
   - FY+1 年度 EPS 預測
   - 營收增長率假設
   - 營運效率/淨利率假設

2. **目標本益比設定**
   - 產業平均 P/E 參考
   - 成長潛力調整
   - 目標 P/E 區間

3. **目標價預估**
   - 12 個月樂觀/保守目標價
   - 保守情境：券商後 25% 或中位數

4. **券商預測彙整（新增）**
   - Top 10 主要券商預測（若有）
   - 表格格式呈現

#### 券商預測表格格式

```markdown
### [標題]

[估值邏輯說明段落]

#### 主要券商預測彙整

| 券商/機構 | FY2027 EPS | 目標價 | 發布日期 |
|----------|-----------|--------|---------|
| XXX證券 | $X.XX | $XX.XX | YYYY/MM/DD |
| YYY投顧 | $X.XX | $XX.XX | YYYY/MM/DD |
| ...      | ...       | ...    | ...        |

**資料來源**: Morningstar Analyst Research  
**註**: 若資料不足10家，僅列出實際可得數量
```

#### 數據來源標註要求

- 明確說明「華爾街共識」或「券商中位數」
- 註明數據截止日期
- 標註資料來源（如 Morningstar）

## 💰 區塊 3：投資建議（250-300字）

### 結構要求

| 項目 | 要求 |
|-----|------|
| 總字數 | 250-300字 |
| 段落數 | 2段 |
| 第1段 | 估值邏輯（150-180字）|
| 第2段 | 評等建議（100-120字）|

### 禁止寫法（理論外露）

❌ "我們採用 DCF 模型進行估值..."  
❌ "根據均值回歸理論..."  
❌ "依據 Gordon 成長模型..."

### 推薦寫法（自然表達）

✅ "目標價基於 2027 年盈利預期與合理估值倍數..."  
✅ "採用前瞻性估值方法，重點放在未來增長潛力..."  
✅ "維持本益比水準隱含長期增長率約 X%..."

### 第1段結構（估值邏輯）

```
參考[數據源]分析師於[日期]更新的公允價值[目標價]。
[估值方法] + [關鍵假設] + [倍數水準] + [隱含增長率與折現率]
```

### 第2段結構（評等建議）

```
當前股價[價格]相對公允價值[空間]，[評等決策]。
主要催化劑：(1) [項目1]; (2) [項目2]; (3) [項目3]。
投資人需留意[風險1]、[風險2]。
```

## ⚠️ 區塊 4：投資風險

### 風險數量規範

| 不確定性評級 | 上行風險 | 下行風險 | 總計 |
|------------|---------|---------|-----|
| Low/Medium | 3項 | 3項 | 6項 |
| High | 4項 | 5項 | 9項 |
| Very High | 4項 | 6項 | 10項 |

### 每項風險格式

```yaml
字數: 120-150字
標題: 自然語言描述（非編號）
結構:
  - 情境描述（若發生X）
  - 影響機制（將導致Y）
  - 量化影響（營收/利潤Z%）
  - 時間框架（何時發生）
```

### 標題範例

✅ 推薦：
- "OpenAI 合作超越預期"
- "Nvidia 主導地位持續強化"
- "大型客戶自研晶片加速"

❌ 禁止：
- "上行風險1"（編號式）
- "競爭風險：市場份額"（理論式）

### 風險內容模板

```
**[自然標題]**。若 [情境X]，[影響機制Y]，[量化影響：營收/
利潤約 Z%]，[時間框架：預期在何時]。
```

## 📚 重要提醒

### 自然語氣原則

1. **理論指導分析，報告用自然語言**
   - 內部思考可用理論框架
   - 報告絕不出現理論術語

2. **標題風格**
   - 陳述式（非疑問句）
   - 包含業務關鍵詞
   - 避免編號與制式表達

3. **數據呈現**
   - 所有數據必須有來源標註
   - 時間點明確（截至日期）
   - 缺失數據標註 "N/A"

### 格式控制

1. **字數嚴格限制**
   - 各區塊字數不可超過上限
   - 不足下限需補充內容

2. **段落優先於列點**
   - 散文段落為主
   - 僅券商預測表使用表格

3. **中英文標點**
   - 中文內容用全形標點
   - 英文/數字用半形
   - 數字與單位間有空格
