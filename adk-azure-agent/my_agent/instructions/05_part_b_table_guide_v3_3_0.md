## 🎯 Part B 概述

### 報告定位

Part B 是**一頁式快速參考表格**：
- 目標讀者：所有投資人（專業與一般投資人）
- 閱讀時間：2-3 分鐘
- 格式：表格化呈現，便於快速查閱
- 風格：資訊密集、重點濃縮

### 核心要求

```yaml
內容定位:
  - 從 Part A 提煉核心資訊
  - 不重複 Part A 的詳細論述
  - 焦點內容必須恰好 4 點

格式要求:
  - 表格化呈現
  - 數據對齊
  - 缺失數據標註 "N/A"
  - 不可留空白

字數控制:
  - 焦點內容：每點 60-80 字
  - 免責聲明：標準格式
```

## 🎯 子區塊 3.2：焦點內容（必須4點）

### 核心要求

```yaml
數量: 必須恰好 4 點（不多不少）
每點結構:
  - 標題：一句話（10-15字）
  - 內容：數據支撐 + 影響說明
  - 字數：60-80 字

提取原則:
  - 從 Part A 核心論點提煉
  - 更簡潔，但保留關鍵數據
  - 不重複 Part A 的詳細論述
```

### 每點的結構範本

```markdown
1. **[標題：一句話描述核心論點]**
   - [關鍵數據1]
   - [關鍵數據2]
   - [影響說明]
```

### 完整範例（AMD）

```markdown
## 焦點內容

1. **OpenAI 戰略合作推動 AI GPU 爆發性增長**
   - AI GPU 營收從 2024 年 US$50 億快速增長至 2027 年預估 US$310 億
   - 2026 下半年開始供應 MI450 系列 GPU 給 OpenAI
   - 2025-2029 年複合增長率預估 62%

2. **x86 CPU 持續從 Intel 奪取市佔率**
   - 受惠 Intel 製程技術困境與 chiplet 策略優勢
   - 客戶端業務預估 17% 年複合增長率、資料中心 CPU 預估 42%
   - Intel 若在 18A 製程成功翻身可能減緩市佔率增長

3. **Nvidia 主導與軟體能力差距是最大挑戰**
   - CUDA 生態系統護城河深厚，ROCm 軟體平台需大幅提升
   - AI 模型遷移成本高，限制市佔率提升速度
   - 大型客戶自研 ASIC 壓縮可服務市場

4. **估值反映高增長預期但不確定性極高**
   - 2025 年 68 倍本益比看似昂貴，但 2027 年降至 26 倍已趨合理
   - 營業利益率預估從 24% 擴張至 36%
   - Very High 不確定性評級反映 AI 市場快速變化

---

[字數統計]
第1點: 76字 ✅
第2點: 71字 ✅
第3點: 68字 ✅
第4點: 72字 ✅
```

### 焦點內容的提煉邏輯

```yaml
Part A 子章節 → Part B 焦點點

Part A 第1章（400字）：AI GPU 業務迎來爆發性增長
  ↓ 提煉
Part B 第1點（76字）：OpenAI 戰略合作推動 AI GPU 爆發性增長

Part A 第2章（400字）：x86 CPU 持續從 Intel 奪取市佔率
  ↓ 提煉
Part B 第2點（71字）：x86 CPU 持續從 Intel 奪取市佔率

Part A 第3章（400字）：軟體能力與客戶集中是主要挑戰
  ↓ 提煉
Part B 第3點（68字）：Nvidia 主導與軟體能力差距是最大挑戰

Part A 投資建議（280字）：估值分析
  ↓ 提煉
Part B 第4點（72字）：估值反映高增長預期但不確定性極高
```

### 常見錯誤

#### ❌ 錯誤 1：超過 4 點

```markdown
## 焦點內容

1. **OpenAI 合作推動增長**
   ...
2. **Intel 市佔率提升**
   ...
3. **軟體能力差距**
   ...
4. **估值分析**
   ...
5. **風險提示**  ← 錯誤：第5點，應該只有4點
   ...
```

#### ❌ 錯誤 2：字數超過上限（95字）

```markdown
1. **OpenAI 戰略合作推動 AI GPU 爆發性增長**
   - AI GPU 營收從 2024 年 US$50 億快速增長至 2027 年預估 
     US$310 億，年複合增長率高達 62%
   - 2026 下半年開始供應最新的 MI450 系列 GPU 產品給 OpenAI，
     這是公司歷史上最重要的戰略合作
   - 分析師大幅上調預測，對未來增長充滿信心
```

**問題**：字數 95 字，超過上限（80字）

#### ❌ 錯誤 3：缺乏具體數據

```markdown
1. **AI GPU 業務快速增長**
   - AI GPU 業務增長強勁
   - 與 OpenAI 建立合作關係
   - 未來前景看好
```

**問題**：沒有具體數據，過於空泛
